# Recipe AId

This is a sample README for the Recipe AId project. It allows users to create, manage, and share recipes.

To use this application, simply follow these instructions: 

1. Clone the repository: git clone https://github.com/StuehlerM/recipeAId.git
2. Navigate into the directory: cd recipeAId
3. Install the dependencies: npm install

If you find any issues, please feel free to open an issue or submit a pull request.
